# בוזיאל מבשל – ספר מתכונים מסודר

This repository contains a personal cookbook with organized recipes in JSON format.

## About

This cookbook contains various recipes including:
- Main dishes (chicken, beef, fish)
- Side dishes
- Salads
- Pickles and preserves
- Sauces and spreads
- Vegan and vegetarian options
- Asian cuisine

The cookbook is managed according to consistent rules, with each recipe having a category, name, language, diet type, equipment, description, ingredients, steps, and tips.

## Contributing

We welcome contributions from the community! If you have a recipe to share or suggestions for improvements, please:

1. **Create an issue** to discuss your recipe or suggestion
2. Follow the existing recipe format in `cookbook.json`
3. Ensure recipes include all necessary details (ingredients, steps, equipment, etc.)

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed contribution guidelines.

## Repository Structure

- `cookbook.json` - Main cookbook file containing all recipes in structured JSON format

## Issues

Issues are public and anyone can create them. Use issues to:
- Suggest new recipes
- Report errors in existing recipes
- Propose improvements to recipe organization
- Ask questions about recipes

Please use the issue templates when creating new issues to ensure all necessary information is provided.
